== -l / --lookup

Searches for QUERY as a filename in all installed package file lists and
skips all other operations.

----
$ sbbdep --quiet -l openssl.cnf
 dovecot-2.3.21.1-x86_64-7: usr/doc/dovecot-2.3.21.1/dovecot-openssl.cnf
 stunnel-5.77-x86_64-1: etc/stunnel/openssl.cnf.new
 ppp-2.5.2-x86_64-1: etc/ppp/openssl.cnf
 openvpn-2.7.0-x86_64-1: usr/doc/openvpn-2.7.0/sample-keys/openssl.cnf
 openssl-solibs-3.5.5-x86_64-1: etc/ssl/openssl.cnf.new
 openssl-3.5.5-x86_64-1: etc/ssl/openssl.cnf.new
 dracut-110-x86_64-1: usr/lib64/dracut/modules.d/11fips/openssl.cnf
----

== --bdtree

Prints a hierarchical binary dependency tree of a dynamic linked file.

----
$ sbbdep --quiet --bdtree /usr/lib64/libssl.so.3

/lib64/libssl.so.3
|----| libcrypto.so.3 => /lib64/libcrypto.so.3
|    | (openssl-solibs-3.5.5-x86_64-1, openssl-3.5.5-x86_64-1)
|    |----| libz.so.1 => /lib64/libz.so.1 => /lib64/libz.so.1.3.2
|    |    | (zlib-1.3.2-x86_64-1, aaa_libraries-15.1-x86_64-49)
|----| libz.so.1 => /lib64/libz.so.1 => /lib64/libz.so.1.3.2
|    | (zlib-1.3.2-x86_64-1, aaa_libraries-15.1-x86_64-49)
used by all:
|    |----| libc.so.6 => /lib64/libc.so.6 => /lib64/libc-2.42.so
|    |    | (glibc-2.42-x86_64-1, aaa_glibc-solibs-2.42-x86_64-1)
|    |    |----| ld-linux-x86-64.so.2 => /lib64/ld-linux-x86-64.so.2
|    |    |    | (glibc-2.42-x86_64-1, aaa_glibc-solibs-2.42-x86_64-1)
----

Note: the output can become very large when used with a package rather than
a single file.
